# Mobile-Sales-Analysis-using-PowerBI-
Detailed analysis of mobile sales with respect to its brand, area of selling, etc.


Project Overview
This project provides a comprehensive analysis of mobile phone sales using Power BI.  
It highlights key trends, customer preferences, brand performance, revenue distribution, top-selling models, and region-wise sales patterns.  
The dashboard helps businesses make data-driven decisions related to inventory planning, marketing strategy, and product pricing.

---

Dataset Information
The dataset contains:
- Mobile brand  
- Model  
- Price  
- Sales units  
- Revenue  
- Date of sale  
- Region / Store location  
- Customer category (optional)

---

 Objectives of the Dashboard
- Identify top-performing mobile brands  
- Analyze monthly and yearly sales trends  
- Compare product pricing vs sales  
- Determine region-wise demand  
- Track revenue contribution from different models  
- Understand customer purchasing patterns  

---

 Key Insights From Dashboard
- Best-selling mobile brand and model  
- Sales trend over time (increasing/decreasing)  
- Price segments generating highest revenue  
- Region/store with maximum sales  
- Contribution breakdown by brand  
- High-demand vs low-demand categories  

---

Dashboard Features
- Interactive slicers (Brand, Region, Year, Price Range)  
- Dynamic KPIs for Sales, Revenue & Units Sold  
- Bar charts, line charts, donut charts  
- Ranking of top 10 models  
- Region-wise heatmap visualization  
- Clean and modern Power BI UI  

---

Tools & Technologies Used
